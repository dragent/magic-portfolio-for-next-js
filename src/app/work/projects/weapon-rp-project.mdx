# Weapon RP Project

Projet d'entraînement à l'utilisation de l'interface d'administration Symfony dans un contexte fictif de serveur RP.

## Technologies utilisées

- **Symfony** (Framework PHP)
- **EasyAdmin** (Interface d'administration)

## Fonctionnalités

- Mise en place et personnalisation d'un backoffice avec EasyAdmin
- Gestion de rôles, entités et permissions
- Structure propre pour l'administration d'un univers RP (personnages, armes, etc.)
- Interface d'administration intuitive

## Statut

Projet terminé (2023)
